   - Icon: User group/people 24x24px
   - Touch Target: Full width of tab area
   - Label: "Contacts" 10px below icon, Inter Regular 10px
   - Color: Medium Gray (#8E8E9A) inactive → Primary Green (#0BBF7D) active
   - Action: Navigate to Contact List View

   - Touch Target: Full width of tab area
   - Label: "Wallet" 10px below icon, Inter Regular 10px
   - Color: Medium Gray (#8E8E9A) inactive → Primary Green (#0BBF7D) active
   - Action: Navigate to Wallet Home

4. SETTINGS TAB (Fourth):
   - Position: Rightmost (25% width)
   - Icon: Gear/Cog 24x24px
   - Touch Target: Full width of tab area
   - Label: "Settings" 10px below icon, Inter Regular 10px
   - Color: Medium Gray (#8E8E9A) inactive → Primary Green (#0BBF7D) active
   - Action: Navigate to Settings Home

• Active tab: Primary Green (#0BBF7D) icon and text
• Inactive tabs: Medium Gray (#8E8E9A) icon and text
• Active indicator: Optional 3px green line at top of tab
• Smooth transition: 200ms ease-in-out

FLOATING ACTION BUTTON (FAB) - OVERLAY:
In addition to the bottom navigation, some pages (Dashboard, Contact List) 
include a FLOATING ACTION BUTTON that overlays the bottom navigation:
• Size: 56x56px
• Position: Bottom-right, 16px from right edge, 96px from bottom (above nav)
• Background: Primary Green (#0BBF7D)
• Icon: Plus (+) 24px, White
• Shadow: 0 6px 16px rgba(0, 168, 107, 0.3)
• Action: Opens quick action menu for adding contacts
• Sub-actions: Add Contact, Scan Card, QR Code, Manual Entry

Usage Logic:
• Visible on: Home Dashboard, Contact List, Wallet Home, Settings Home
• Hidden on: Full-screen flows (Auth, Onboarding, Scanner), Deep inner pages
• Consistent across all main module pages

CRITICAL FOOTER DESIGN SUMMARY:
-------------------------------
The Bottom Navigation is a STANDARD 4-TAB NAVIGATION BAR with equal spacing:
  [Home] [Contacts] [Wallet] [Settings]

Each tab shows an icon with a label below it. The active tab is highlighted
in Primary Green, while inactive tabs appear in Medium Gray. This provides
consistent navigation across ALL major sections of the application.

Additional floating action buttons (like the FAB) may overlay this navigation
on specific pages, but the 4-tab bottom navigation remains the PRIMARY and
CONSISTENT navigation mechanism throughout the entire application.
